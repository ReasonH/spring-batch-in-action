## Spring-Batch Tutorial

스프링 배치 어플리케이션의 개념에 대해 학습하고 DB 연동 및 간단한 배치 어플리케이션을 경험해봅니다.  
해당 Repository 의 정리 내용은
[기억보단 기록을](https://jojoldu.tistory.com/category/Spring%20Batch)를 읽고 참조하고 있습니다.  

실습은 작성 시점의 구현환경에 맞게 변경했습니다.

- Spring Boot 2.3.0
- Spring Batch
- DB: H2 / MySQL (8+)
- JPA
